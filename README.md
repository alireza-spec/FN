# 🎬 Family Night

### A single-file, self-hosted movie & TV discovery app / یک اپلیکیشن تک‌فایلی برای کشف فیلم و سریال

**🔗 Live Demo:** [alireza-spec.github.io/FN](https://alireza-spec.github.io/FN/)

*Built entirely as one portable HTML file — no build tools, no backend, no database.*
*کاملاً در قالب یک فایل HTML — بدون نیاز به بیلد، بک‌اند یا دیتابیس.*

## 📖 About / درباره‌ی پروژه

**English:**
Family Night is a personal, feature-rich movie and TV show discovery app powered by [TMDB](https://www.themoviedb.org/). It was built as a single self-contained `index.html` file and deployed for free on GitHub Pages — designed for one person's love of cinema, and shaped entirely through iterative, hands-on development.

**فارسی:**
فمیلی‌نایت یک اپلیکیشن شخصی و پرقابلیت برای کشف فیلم و سریال است که با API سایت [TMDB](https://www.themoviedb.org/) کار می‌کند. این پروژه به‌صورت کامل در یک فایل `index.html` ساخته شده و به‌رایگان روی GitHub Pages میزبانی می‌شود — ساخته‌شده از دل علاقه‌ی شخصی به سینما، و قدم‌به‌قدم و کاملاً دستی توسعه داده شده است.

## ✨ Features / امکانات

### 🎞️ Discovery & Browsing / کشف و مرور
- Real-time movie & TV data from TMDB, with full bilingual (Persian/English) UI throughout
- Smart **Curator's Pick** and **"Pick a Movie for Me"** floating button with a custom scoring algorithm
- **Quote of the Day** — a curated database of 220+ film dialogues
- **Smart Similar Titles** — a from-scratch recommendation engine that scores suggestions by *shared genres, cast, director/creator, and keywords* (no randomness, no generic "TMDB similar" noise)
- **Compare** — pick any two titles via live bilingual search and see them compared side-by-side
- **Collections** — real, TMDB-verified movie franchises (Godfather, Harry Potter, LOTR, Marvel, and 200+ more) shown with a "stacked" visual, each opening to a full grid of its entries
- Country filter across 144 countries, expanded genre/category browsing

### 📺 Watching / تماشا
- Multiple streaming server options with a tested, pruned server list
- **Free & Legal Downloads** — a growing archive of genuinely Public Domain movies & classic TV, sourced live from Archive.org and the Library of Congress, matched against real TMDB data (accurate covers, ratings, years — never placeholders)
- **Family Night Explore** — an Instagram-Explore-style feed of cinematic edits, scenes, and fan clips pulled live from YouTube, Dailymotion, Vimeo, Tumblr, and Klipy, complete with a Reels-style vertical player (tap to pause, double-tap to like, swipe to browse)

### 🧠 Personalization / شخصی‌سازی
- AI chat assistant on every title's page to ask questions about it
- Personality analysis based on viewing taste
- Watch History with real posters, a movies/series/documentaries breakdown, and per-item or full history deletion
- Favorites, Watchlist, and quizzes/tests about film trivia
- 12 built-in visual themes (default: **Soft Dark Gray**) that reshape the entire app's color palette, with a clear active-theme indicator

### 🌐 Design / طراحی
- Fully responsive, mobile-first interface
- Centered, elevated Home button in the bottom navigation
- Settings accessible from a header icon and the side menu on every screen
- Every UI element — from banners to loaders — respects the selected theme's colors

## 🚀 Getting Started / راه‌اندازی

1. Clone or download this repository.
2. Open `index.html` directly in a browser — or deploy the repo to **GitHub Pages** for free hosting.
3. That's it. No `npm install`, no build command.

```bash
git clone https://github.com/alireza-spec/FN.git
cd FN
# open index.html in your browser, or push to GitHub Pages
```

> **Note:** Some features (the Explore feed's YouTube/Vimeo/Tumblr/Klipy sources) require you to add your own free API keys inside the code where indicated — Dailymotion and the core app work with no setup at all.
