# 🎬 Family Night Ultimate

یک اپلیکیشن وب فارسی برای انتخاب فیلم و سریال خانوادگی — با امکانات کامل.

## ✨ امکانات

- 🎲 پیشنهاد فیلم تصادفی
- 🎭 فیلتر بر اساس حال‌وهوا (Mood Picker)
- ⭐ امتیازدهی شخصی
- 📊 آمار سینمایی
- 🤖 دستیار هوشمند AI
- 📋 Watchlist شخصی
- 🔍 جستجو در فیلم‌ها و سریال‌ها
- 🌙 حالت سینماتیک

## 🚀 راه‌اندازی روی GitHub Pages

1. این ریپو را Fork یا Clone کنید
2. به **Settings** ریپو بروید
3. از منوی سمت چپ **Pages** را انتخاب کنید
4. در بخش **Source** گزینه `Deploy from a branch` را بزنید
5. Branch را روی `main` و فولدر را روی `/ (root)` بگذارید
6. **Save** کنید — بعد از چند دقیقه سایت آماده است!

## 📁 ساختار پروژه

```
family-night/
├── index.html          # فایل اصلی اپلیکیشن
├── README.md           # این فایل
├── .nojekyll           # برای GitHub Pages (مهم)
└── assets/             # فایل‌های جانبی (آینده)
    ├── css/
    ├── js/
    └── icons/
```
